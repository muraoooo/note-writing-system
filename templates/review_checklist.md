# 公開前 レビューチェックリスト

公開ボタンを押す前に、必ず全項目をチェック。1つでも欠けたら直してから公開。

## ① タイトル(最重要)

- [ ] 26〜32文字に収まっている
- [ ] 検索キーワードが前半に置かれている
- [ ] 数字が含まれている(「3つの」「30分で」など)
- [ ] 【勉強メモ】など自分用感を出すワードを含めていない
- [ ] 括弧書きでオチを言っていない
- [ ] 候補を3案以上作って一番尖ったものを選んだ

## ② サムネイル

- [ ] 1280×670 で出力した
- [ ] スマホサイズで文字が読める
- [ ] 連載テンプレと統一されている
- [ ] デフォルトの自動生成画像のままではない

## ③ リード文(冒頭3行)

- [ ] 1文目で読者の悩みまたは結論が出ている
- [ ] 自己紹介(「こんにちは」)から始まっていない
- [ ] 80〜150文字に収まっている
- [ ] 「続きを読む」の上に置く文章として釣れる

## ④ 見出し構成

- [ ] H2が7〜10個ある
- [ ] 各見出しに動詞または結論が含まれる
- [ ] 見出しだけ並べて読んで内容が伝わる
- [ ] 「はじめに」「最後に」だけで本文見出しがない状態ではない

## ⑤ 本文

- [ ] 1段落が3行以内
- [ ] 1文が60文字以内
- [ ] 500文字ごとに視覚要素(画像/コード/表)がある
- [ ] 太字の乱用がない(1段落に1箇所まで)
- [ ] 文体が「ですます」または「である」で統一
- [ ] 専門用語の初出に補足がある

## ⑥ 視覚要素

- [ ] 画像が3個以上
- [ ] コードブロックがコピペで動く状態
- [ ] 表で比較・整理されている箇所がある
- [ ] 画像にキャプション(alt相当)が付いている

## ⑦ 締め

- [ ] 要約(3行)がある
- [ ] 内部リンク(次に読むおすすめ)が3本ある
- [ ] CTA(スキ / フォロー / コメント)が1つ明示されている
- [ ] 連載サイン文が入っている

## ⑧ ハッシュタグ

- [ ] タグ数が5〜7個
- [ ] 大規模タグ1〜2個 + 中規模2〜3個 + ニッチ1〜2個
- [ ] シリーズ共通タグが含まれる
- [ ] 関係ないバズワードタグを入れていない

## ⑨ SEO

- [ ] メインキーワードがタイトル前半
- [ ] リードにキーワードが含まれる
- [ ] H2の3つ以上にキーワード派生形
- [ ] 公式ドキュメントへの外部リンクがある

## ⑩ 公開タイミング

- [ ] 平日 朝7-8時 or 夜21-23時 を狙っている
- [ ] 公開後にTwitter/X告知の準備ができている
- [ ] マガジンへの登録準備ができている

- [ ] ビューが公開3時間で50以上ある(初速指標)
- [ ] スキ率が1%を下回っていないか
- [ ] コメントが来たら24時間以内に返信した
- [ ] 想定外のキーワードで流入していたら、タグを微修正

- [ ] マガジンに登録した
- [ ] 反応が悪ければタイトル/リードのABテストを実施
